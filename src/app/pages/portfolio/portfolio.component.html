ESport

This project focuses on the efficient management of sports, specifically handling sport events, timetables, and related tasks. In addition to its core functionalities, the system offers advanced features for live streaming and recording of sport events. I was responsible for designing the databases and the overall architecture of the backend system. Additionally, I developed the video streaming functionality, which included implementing one-to-many video conferences and stream recording. This was achieved by utilizing Kurento Media Server.

Responsibilities
  • Implementing streaming functionality
  • Implementing timetable feature
  • Designing database
  • Setting up CI/CD with infrastructure for deploy
  • Setting up testing environment
Technologies

C# .Net Core ASP.NET .Net Framework Entity Framework SignalR MSSQL Docker Angular Git Kurento Media Server Ocelot
